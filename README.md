## Overview

This repository contains the architecture, product requirements, workflow design, data model, UX strategy, and implementation decisions for the AGI&M Professional Battery Sourcing and Quotation Workflow Platform.

This is a long-term, reusable B2B procurement platform. The first vertical is **drone lithium battery cells**. Future verticals include robotics, AGV, marine, industrial ESS, and eVTOL battery applications.
